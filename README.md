# 🔮 塔罗牌占卜网站 v3.5

一个功能完整的塔罗牌占卜网站，采用纯前端技术实现，包含完整的78张塔罗牌（22张大阿尔卡纳 + 56张小阿尔卡纳），支持多种占卜方式。

## 🌟 功能特性

### 核心占卜功能
- ⚡ **快速占卜**：直接选择占卜类型，立即开始占卜
- 🧭 **引导占卜**：输入问题 → 确认 → 选择占卜类型的完整引导流程
- ✨ **多种占卜方式**：爱情、事业、未来、财运、健康、人际关系、随机指引
- ✝️ **凯尔特十字牌阵**：最经典的10张牌深度占卜

### 辅助功能
- 🌅 **每日一牌**：每天抽取一张专属塔罗牌，获得今日指引
- 🎯 **是/否占卜**：快速获得是或否的答案
- 📖 **塔罗牌图鉴**：浏览全部78张塔罗牌的详细信息，按组别分类展示
- 📜 **历史记录**：自动保存占卜历史（最多50条），随时回顾

### 学习中心 🎓
- 📚 **牌义百科**：完整的78张塔罗牌详细解读，支持搜索和筛选
- 🎮 **互动练习**：
  - 认牌练习：通过符号识别塔罗牌
  - 牌义测试：根据场景选择合适的牌
  - 实时统计：记录练习次数和准确率
- 📖 **塔罗课程**：
  - 新手入门：塔罗简介、结构、占卜方法、基础牌阵
  - 进阶技巧：解读技巧、牌组合、直觉培养、高级牌阵

### 用户体验
- 🔮 **完整引导流程**：洗牌进度条、切牌提示、抽牌计数，每步都有明确指引
- 🔊 **音效系统**：洗牌、翻牌、选择等音效增强体验
- 📱 **分享功能**：支持分享占卜结果到社交平台
- 🎨 **正位/逆位**：完整的正逆位解读系统，每张牌6个维度解读
- 💫 **精美动画**：流畅的卡牌动画和页面切换效果

## 🚀 技术栈

- **HTML5**：语义化标签
- **CSS3**：Flexbox、Grid、动画、渐变
- **JavaScript (ES6+)**：模块化、箭头函数、解构赋值

## 📦 模块说明

- **tarot-cards.js**：包含完整78张塔罗牌数据

## 🎯 使用方法

### 本地运行

1. **直接打开**：在浏览器中打开 `index.html` 即可使用
   
2. **本地服务器**（推荐，避免CORS问题）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 或使用Node.js
   npx http-server
   
   # 或使用PHP
   php -S localhost:8000
   ```
   然后访问 `http://localhost:8000`

### 占卜流程

#### 快速占卜（推荐新手）
1. **选择占卜类型**：直接点击爱情、事业、未来等占卜类型
2. **洗牌**：系统自动洗牌（带进度条显示）
3. **切牌**：凭直觉点击左、中、右任一位置切牌
4. **抽牌**：从展开的牌中凭直觉抽取牌（显示已选数量）
5. **查看结果**：查看每张牌的解读和整体总结
6. **分享/重新占卜**：可以分享结果或返回首页

#### 引导占卜（更精准）
1. **输入问题**：在问题输入页面详细描述你的问题
2. **确认问题**：点击"开始占卜"确认
3. **选择占卜类型**：根据问题选择合适的占卜类型
4. **后续流程**：与快速占卜相同（洗牌→切牌→抽牌→结果）

## 🎨 自定义

### 修改主题颜色
在 `css/main.css` 中修改以下样式：
```css
body {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### 添加新的占卜类型
1. 在 `index.html` 的占卜选项区域添加新按钮
2. 在 `divination.js` 的 `spreadConfig` 中添加配置：
   ```javascript
   newtype: {
       title: '新类型占卜',
       positions: ['位置1', '位置2', '位置3'],
       dimension: 'future'  // 使用的解读维度
   }
   ```
3. 在 `getSummary()` 函数中添加总结文案
4. 在 `storage.js` 的 `getSpreadTypeName()` 中添加类型名称

### 修改音效
在 `audio.js` 中调整音效参数：
```javascript
const soundConfig = {
    shuffle: { frequency: 200, duration: 0.1 },
    flip: { frequency: 400, duration: 0.15 },
    // ...
};
```
