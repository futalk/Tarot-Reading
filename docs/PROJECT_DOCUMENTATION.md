## 🎯 项目概述

### 项目简介

一个功能完整的塔罗牌占卜网站，采用纯前端技术实现，包含完整的78张塔罗牌（22张大阿尔卡纳 + 56张小阿尔卡纳），支持多种占卜方式。

### 技术栈

- **HTML5**：语义化标签
- **CSS3**：Flexbox、Grid、动画、渐变
- **JavaScript (ES6+)**：模块化、箭头函数、解构赋值
- **Web APIs**：LocalStorage、Web Audio API、Web Share API、Clipboard API

### 项目特点

- ✅ 纯前端实现，无需后端
- ✅ 完整的78张塔罗牌数据
- ✅ 多种占卜方式
- ✅ 智能学习系统
- ✅ 响应式设计
- ✅ 离线可用

## 🌟 核心功能

### 1. 占卜功能

#### 快速占卜
- ⚡ 直接选择占卜类型，立即开始
- ✨ 支持：爱情、事业、未来、财运、健康、人际关系、随机指引
- ✝️ 凯尔特十字牌阵（10张牌深度占卜）

#### 引导占卜
- 🧭 输入问题 → 确认 → 选择占卜类型
- 📝 完整的引导流程
- 🎯 更精准的占卜结果

#### 特殊占卜
- 🌅 每日一牌：每天抽取一张专属塔罗牌
- 🎯 是/否占卜：快速获得是或否的答案

### 2. 学习中心

#### 牌义百科
- 📚 完整的78张塔罗牌详细解读
- 🔍 支持搜索和筛选
- 📖 正位和逆位各6个维度解读

#### 互动练习
- 🎮 认牌练习：通过符号识别塔罗牌
- 🎯 牌义测试：根据场景选择合适的牌
- 📊 实时统计：记录练习次数和准确率

#### 塔罗课程
- 📖 新手入门：塔罗简介、结构、占卜方法、基础牌阵
- 🎓 进阶技巧：解读技巧、牌组合、直觉培养、高级牌阵

### 3. 辅助功能

- 📖 **塔罗牌图鉴**：浏览全部78张塔罗牌，按组别分类展示
- 📜 **历史记录**：自动保存占卜历史（最多50条）
- 📱 **分享功能**：支持分享占卜结果到社交平台
- 🔊 **音效系统**：洗牌、翻牌、选择等音效增强体验

### 4. 高级功能（v3.4+）

#### 扩展牌组合数据库
- 📊 150+个牌组合
- 🎯 包含meaning、theme、advice、context
- 🔍 支持组合查询和统计

#### 情境化解读系统
- 🎯 10个主情境 + 40+个子情境
- 📝 200+个关键词覆盖
- 🔍 智能情境识别

#### 智能学习系统
- 📊 用户反馈收集
- 🎯 个性化推荐
- 📈 趋势分析和报告生成
- 💾 LocalStorage存储（最多500条记录）
