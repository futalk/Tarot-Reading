## 💻 安装和运行

### 方法1：直接打开（最简单）

1. 下载项目文件
2. 在浏览器中打开 `index.html`
3. 开始使用！

### 方法2：本地服务器（推荐）

避免CORS问题，使用本地服务器：

```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx http-server

# 或使用PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`

## 🔮 占卜流程

### 快速占卜（推荐新手）

1. **选择占卜类型**
   - 点击首页的占卜类型按钮
   - 例如：爱情、事业、未来等

2. **洗牌阶段**
   - 系统自动洗牌
   - 显示进度条和提示

3. **切牌阶段**
   - 凭直觉点击左、中、右任一位置
   - 切牌会影响占卜结果

4. **抽牌阶段**
   - 从展开的牌中凭直觉抽取
   - 显示已选数量

5. **查看结果**
   - 查看每张牌的解读
   - 查看整体总结
   - 可以分享或重新占卜

### 引导占卜（更精准）

1. **输入问题**
   - 在问题输入页面详细描述问题
   - 越具体越好

2. **确认问题**
   - 点击"开始占卜"确认

3. **选择占卜类型**
   - 根据问题选择合适的类型

4. **后续流程**
   - 与快速占卜相同
   - 洗牌 → 切牌 → 抽牌 → 结果

### 凯尔特十字牌阵

最经典的10张牌深度占卜：

1. 选择"凯尔特十字"牌阵
2. 依次抽取10张牌
3. 每张牌代表不同位置和含义
4. 获得全面深入的解读

## ❓ 常见问题

### Q1: 为什么直接打开HTML文件会有问题？

**A**: 浏览器的CORS安全策略可能阻止某些功能。建议使用本地服务器。

### Q2: 占卜结果可以保存吗？

**A**: 可以！系统自动保存最近50条占卜记录到历史记录中。

### Q3: 每日一牌可以抽多次吗？

**A**: 不可以。每天只能抽取一次，第二天才能重新抽取。

### Q4: 如何分享占卜结果？

**A**: 占卜结束后，点击"分享结果"按钮，可以分享到社交平台或复制到剪贴板。

### Q5: 音效可以关闭吗？

**A**: 可以。点击右上角的音效按钮即可开关音效。

### Q6: 支持哪些浏览器？

**A**: 支持所有现代浏览器：
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- 移动端浏览器

### Q7: 占卜结果准确吗？

**A**: 塔罗牌占卜仅供娱乐参考，命运掌握在自己手中。

### Q8: 可以自定义牌阵吗？

**A**: 目前不支持，但在未来规划中。

### Q9: 数据会丢失吗？

**A**: 数据保存在浏览器的LocalStorage中，清除浏览器数据会导致丢失。建议定期导出（未来功能）。

### Q10: 如何清除历史记录？

**A**: 进入"历史记录"页面，点击"清空历史"按钮。

## 🎯 使用技巧

### 占卜技巧
