## 🔮 核心占卜功能

### 快速占卜

#### 支持的占卜类型

| 类型 | 说明 | 牌数 | 适用场景 |
|------|------|------|----------|
| 💕 爱情 | 感情问题占卜 | 3张 | 恋爱、婚姻、感情困惑 |
| 💼 事业 | 工作事业占卜 | 3张 | 职业发展、工作选择 |
| 🔮 未来 | 未来趋势占卜 | 3张 | 未来走向、发展预测 |
| 💰 财运 | 财富运势占卜 | 3张 | 投资、理财、财运 |
| 🌿 健康 | 健康状况占卜 | 3张 | 身心健康、养生 |
| 🤝 人际关系 | 人际互动占卜 | 3张 | 人际关系、社交 |
| ✨ 随机指引 | 随机指引占卜 | 1张 | 日常指引、灵感 |
| ✝️ 凯尔特十字 | 深度综合占卜 | 10张 | 复杂问题、全面分析 |

#### 占卜流程详解

**1. 洗牌阶段（3秒）**
- 系统随机打乱78张牌
- 显示进度条和动态提示
- 播放洗牌音效

**2. 切牌阶段**
- 选择左、中、右任一位置
- 切牌影响牌组顺序
- 切出的底牌作为"切牌"
- 切牌反映你的心态和能量状态

**3. 抽牌阶段**
- 从展开的牌中凭直觉选择
- 显示已选数量/总数
- 每张牌可能正位或逆位
- 播放翻牌音效

**4. 结果展示**
- 每张牌的详细解读
- 牌组合深度解读
- 整体总结和建议
- 切牌解读

### 引导占卜

#### 使用流程

**步骤1：输入问题**
```
示例问题：
- "我应该接受这份新工作吗？"
- "我和TA的关系会如何发展？"
- "未来三个月我的事业运势如何？"
```

**步骤2：确认问题**
- 检查问题是否清晰
- 点击"开始占卜"

**步骤3：选择占卜类型**
- 根据问题选择合适的类型
- 系统会记住你的问题

**步骤4：占卜流程**
- 与快速占卜相同
- 结果会结合你的问题

#### 问题建议

✅ **好的问题**：
- 具体明确
- 有时间范围
- 可操作性强

❌ **避免的问题**：
- 过于宽泛
- 是非题（用是/否占卜）
- 涉及他人隐私

### 凯尔特十字牌阵

#### 牌位含义

| 位置 | 名称 | 含义 |
|------|------|------|
| 1 | 现状 | 当前的处境和状态 |
| 2 | 挑战 | 面临的障碍或挑战 |
| 3 | 过去 | 过去的影响和根源 |
| 4 | 未来 | 近期的发展趋势 |
| 5 | 目标 | 你的目标和期望 |
| 6 | 潜意识 | 内心深处的想法 |
| 7 | 建议 | 应该采取的行动 |
| 8 | 外部影响 | 环境和他人的影响 |
| 9 | 希望/恐惧 | 你的希望或恐惧 |
| 10 | 结果 | 最终的结果和走向 |

#### 解读技巧

1. **整体观察**
   - 先看整体牌面
   - 注意正逆位比例
   - 观察牌的花色分布

2. **分区解读**
   - 过去（位置3）
   - 现在（位置1、2、6）
   - 未来（位置4、10）
   - 行动（位置5、7、8、9）

3. **牌组合分析**
   - 相邻牌的关系
   - 对立牌的冲突
   - 呼应牌的强化

### 每日一牌

#### 功能说明

- 🌅 每天抽取一张专属塔罗牌
- 📅 使用日期作为随机种子
- 💾 自动保存到历史记录

#### 使用方法

1. 点击"每日一牌"
2. 点击"抽取今日牌"
3. 查看牌的解读
4. 思考今日指引

#### 最佳实践

- 🌄 早晨抽取，作为一天的指引
- 📝 记录感悟和实际发生的事
- 🔄 第二天对比验证
- 📈 长期追踪，总结规律

### 是/否占卜

#### 功能说明

- 🎯 快速获得是或否的答案
- 🃏 单张牌占卜
- ⚡ 适合简单决策

#### 判断规则

**是（正面）**：
- 正位的正面牌
- 例如：太阳、星星、世界等

**否（负面）**：
- 逆位的牌
- 正位的负面牌
- 例如：塔、恶魔等

**可能（中性）**：
- 中性牌
- 需要更多信息

#### 适用场景

✅ **适合**：
- 简单的是非选择
- 快速决策
- 日常小事

❌ **不适合**：
- 复杂问题
- 需要详细分析的情况
- 重大决策

## 🎯 使用建议

### 占卜建议

1. **保持开放心态**
   - 不要带有预设答案
   - 接受各种可能性

2. **专注当下**
   - 占卜时保持专注
   - 避免分心

3. **理性看待**
   - 占卜是参考，不是决定
   - 结合实际情况判断

4. **定期占卜**
   - 不要过度占卜
   - 给自己时间思考和行动

### 学习建议

1. **循序渐进**
   - 从基础开始
   - 逐步深入

2. **多加练习**
   - 使用互动练习
   - 实践中学习

3. **记录总结**
   - 记录占卜结果
   - 总结经验教训

4. **保持好奇**
   - 探索不同牌阵
   - 尝试新的解读方法
