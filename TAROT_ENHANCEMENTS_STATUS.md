## 💡 数据结构模板

为了保持一致性，所有新增的牌都应遵循以下结构：

```javascript
'牌名': {
    deepMeaning: {
        upright: '正位的深层灵性和心理含义，150-200字，触及象征意义、原型能量和人生洞察',
        reversed: '逆位的深层含义和警示，100-150字，反映阻碍或需要调整的地方'
    },
    actionAdvice: {
        upright: [
            '立即行动：今天就可以做的具体行动',
            '本周计划：一周内的短期计划和目标',
            '长期目标：持续发展的方向和习惯培养',
            '心态调整/能量管理：内在工作和心态转变'
        ],
        reversed: [
            '立即停止/反思：需要暂停或改变的行为',
            '本周行动：纠正性措施和调整计划',
            '长期改善：根本性改变和模式打破',
            '寻求支持/平衡：外部资源和平衡恢复'
        ]
    }
}
```
